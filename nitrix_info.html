Installing flatpak

The Good news is that flatpak is already installed

Point to the Flathub Repo to get packages to install

flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo --user

Documentation on the web is located at Documentation

Workflow using Libreoffice as an example
  1. flatpak search libreoffice -- retrieve the application id
  2. flatpak --assumeyes install org.libreoffice.LibreOffice --user
  3. flatpak run org.libreoffice.LibreOffice
Add as an Icon on the doc
